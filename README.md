# Secure Transfer

Encrypted file transfer utility.

Utilitas transfer file terenkripsi.


## Description

Provides encrypted file transfers with compression and checksum verification.

## Deskripsi

Menyediakan transfer file terenkripsi dengan kompresi dan verifikasi checksum.


## Installation

- Navigate to the project folder
- No dependencies required

## Instalasi

- Masuk ke folder proyek
- Tidak perlu dependensi


## Usage

- Encrypt: python secxfer.py encrypt file.txt -p password
- Decrypt: python secxfer.py decrypt file.txt.enc -p password
- Demo: python secxfer.py --demo

## Penggunaan

- Enkripsi: python secxfer.py encrypt file.txt -p password
- Dekripsi: python secxfer.py decrypt file.txt.enc -p password
- Demo: python secxfer.py --demo
